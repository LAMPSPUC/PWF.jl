Quick Start Guide

All functionalities of the PWF.jl package can be used by the following function

using PWF

file = "3bus.pwf"
pwf_dict = parse_file(file; pm = true, add_control_data = false)

The parameter pm indicates if the data should be converted to the PowerModels.jl format. add_control_data is used to inform if additional information regarding control actions should be included in the PowerModels.jl data. For more information about Control Actions visit the ControlPowerFlow.jl documentation

For more information about PowerModels.jl visit the PowerModels documentation
